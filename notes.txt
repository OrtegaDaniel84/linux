
## ðŸŒ Network Interface Naming (Optional)

By default, Debian uses "Predictable Interface Names" (like `enp2s0`). If you prefer the classic naming convention (`eth0`, `wlan0`):

1. **Edit GRUB:**
   vi /etc/default/grub`
   Add "net.ifnames=0 biosdevname=0" to GRUB_CMDLINE_LINUX_DEFAULT.
      GRUB_CMDLINE_LINUX_DEFAULT="quiet net.ifnames=0 biosdevname=0"

2. Update & Apply:
   update-grub

3. Update Networking
   vi /etc/network/interfaces
   replace enp1s0 with eth0
   with vi = :%s/enp1s0/eth0/g

4- Reboot


## External disc

Using and external disc with the Zimablade to get more space and set docker path to this disc

1. Mount the disc with CasaOS
   Check the name and where mount, I use disc and the path is /mnt/disc

2. Change the Docker configuration
   sudo systemctl stop docker docker.socket
   sudo mv /var/lib/docker /mnt/disc/
   echo '{ "data-root": "/mnt/disc/docker" }' | sudo tee /etc/docker/daemon.json > /dev/null
   sudo systemctl daemon-reload && sudo systemctl start docker
   docker info | grep Root
